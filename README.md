# Звіт про проходження навчального курсу



**Посилання:** https://www.kaggle.com/learn/intro-to-machine-learning/course
**Курс:** Intro to Machine Learning

---

## 1. Чому був обраний саме цей курс?

Технології машинного навчання (Machine Learning) сьогодні інтегровані практично в кожну сучасну галузь. Розуміння принципів роботи ML стало не просто перевагою, а необхідною базовою навичкою для будь-якого спеціаліста, який працює з даними. Курс мінімізує суху теорію на користь практики. Кожен новий концепт одразу підкріплюється написанням реального коду, що дозволяє краще закріпити матеріал. Матеріал сфокусований на ключових концепціях базового рівня. Це ідеальний варіант для того, щоб швидко закласти міцний фундамент і отримати перші працюючі результати.

## 2. Чого ви навчились під час проходження курсу?

Під час навчання я пройшов весь базовий життєвий цикл розробки моделі машинного навчання. Зокрема, я опанував такі етапи:
* **Дослідження даних** Використання бібліотеки Pandas для завантаження та аналізу датасетів.

* **Побудова моделі:** Створення моделей на основі дерев рішень (Decision Tree) та випадкового лісу (Random Forest).

* **Валідація:** Використання метрики MAE (Mean Absolute Error) для оцінки точності прогнозів.

## 3. Що було найскладнішим під час проходження курсу?

На початкових етапах найскладнішим було зрозуміти, як саме алгоритм приймає рішення під капотом. Оскільки курс максимально орієнтований на практичне застосування готових інструментів, він практично ігнорує математичну та логічну реалізацію алгоритмів.


## 4. Який матеріал курсу був для вас беззмістовним (найменш корисним)?

Найменш інформативним та занадто спрощеним мені здався блок, присвячений Random Forest. У рамках курсу алгоритм згадується дуже побіжно і подається як структура, яка просто працює краще та стабільніше за звичайне дерево рішень.

Мені забракло глибшого пояснення внутрішньої механіки цього алгоритму. Хотілося б побачити хоча б базовий опис того, як саме будуються численні незалежні дерева, як вони голосують або усереднюють свої результати.
